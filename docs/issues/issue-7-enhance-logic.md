## 背景・目的
幸運コインが貯まり強化ボタンが押せる状態を検出し、強化を実行するロジックを実装する。

## 要件

### やること
- `EnhanceAction` クラスの実装
  - `execute()` → 強化ボタンをタップして強化を実行する
- `ScreenAnalyzer` の `ENHANCE_AVAILABLE` 状態検出をトリガーとする
- タップ後、強化完了ダイアログを検出して閉じる
- 強化実行をログに記録する

### やらないこと
- 強化素材の選択（ボタンを押すだけで自動処理される仕様）
- 幸運コイン残量の数値読み取り

## 完了条件（DoD）
- [ ] `ENHANCE_AVAILABLE` 検出時に `EnhanceAction.execute()` が呼ばれ、強化ボタンがタップされる
- [ ] 強化完了ダイアログが検出されて閉じられる
- [ ] 強化実行がCLIログに「[ENHANCE] 実行」として出力される
- [ ] ダイアログが出ない場合もフリーズしない/#(/#(/#(るとき（タイムアウト処理あり）
