## 背景・目的
ADB操作（スクショ取得・タップ・スワイプ）を各モジュールから統一して呼こ出せるラッパークラスを実装する。
後続のすべての画面操作モジュールがこのクラスを使用する。

## 要件

### やること
- `DeviceController` クラスの実装
  - `screenshot()` → numpy配列でスクリーン画像を返す
  - `tap(x, y)` → 指定座標をタップ
  - `swipe(x1, y1, x2, y2, duration_ms)` → スワイプ（ドラッグに使用）
  - `get_screen_size()` → 解像度を返す
- ADB接続エラー時に例外を発生させてログに記録する
- `config.yaml` から解像度・ADBポートを読み込む

### やらないこと
- 画像認識処理
- ゲームロジック
